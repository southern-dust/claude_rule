## 使用说明

本文档为开发计划模板，使用占位符标记需要替换的内容。

### 占位符说明

| 占位符 | 说明 | 示例 |
|--------|------|------|
| {PROJECT_NAME} | 项目名称 | 敏捷项目管理工具 |
| {UPDATE_DATE} | 更新日期 | 2024-12-25 |
| {PROJECT_STATUS} | 项目状态 | Phase 0/MVP完成 |
| {CURRENT_PRIORITY} | 当前优先级 | GUI界面实现 |
| {PHASE_NAME} | 阶段名称 | Phase 1: 需求分析 |
| {PRIORITY} | 优先级 | 高/中/低 |

### 填写示例

```markdown
# 开发计划

**更新时间**: 2024-12-25
**项目状态**: Phase 5
**当前优先级**: GUI界面实现
```

## 使用说明

### 如何添加新需求？

在本文档末尾按照模板添加：

```markdown
### [待实现] 需求标题

**类型**: 新功能 / 增强 / 优化
**优先级**: 高 / 中 / 低
**提出日期**: YYYY-MM-DD
**描述**: 详细描述需求
```

### 如何报告问题？

在本文档末尾按照模板添加：

```markdown
### [待修复] 问题标题

**类型**: Bug / 错误 / 异常
**严重程度**: 严重 / 中等 / 轻微
**报告日期**: YYYY-MM-DD
**现象**: 问题描述
**复现**: 重现步骤（如有）
```

### Claude会处理吗？

✅ **是的！Claude会在每次对话开始时阅读本文档**

如果你添加了新需求或问题，Claude会：
1. 识别新条目
2. 评估优先级
3. 制定实现计划
4. 开始处理
